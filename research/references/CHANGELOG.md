# 焚诀 Changelog

## V2.3 (2026-07-07) — 投资操作系统
- **新增 Level 0: 市场状态** — Risk-On/Neutral/Off/Crisis 四级，影响仓位上限+产业权重
- **新增 Level 3: 建仓模型** — 每只标的 3 步建仓计划（触发条件+仓位+价位区间）
- **新增 认知差指数** — 市场误解程度，兴发9分(还当它是磷化工)、长电2分(全市场共识)
- **新增 置信度** — 评分有多确定？长电 93% vs 阿科力 55%
- **新增 三级池** — S池(建仓)/A池(等待)/B池(观察)
- **新增 资金风格** — 机构/游资/量化主导判断
- **改名**: 选股系统 → 认知差驱动的产业投资操作系统

## V2.2 (2026-07-07) — 六维模型
- **新增 产业映射树** — 不按行业标签打分，按资金会流向的供应链路径
- **新增 赔率/胜率拆分** — 兴发赔率9胜率7 vs 长电赔率5胜率9
- **新增 兑现周期** — Q3/Q4/H2/8月里程碑
- **新增 失效条件** — 不是跌15%止损，是逻辑失效才删
- **新增 产业雷达** — 先做行业配置(AI材料★★★★★)，再看个股映射

## V2.1 (2026-07-07) — 框架化
- **产业评分从标签改为映射树** — 兴发从5.0→7.2，因为资金沿AI供应链会找它
- **预期兑现独立成维度** — 阿科力控制权变更已兑现，降级
- **权重调整**: 产业趋势35% + 资金流向25% + 机构15% + 融资10% + 预期10%

## V2.0 (2026-07-07) — 初版
- 六维评分: 产业40%、机构20%、融资15%、量化5%、筹码5%、预期10%
- 东方财富/腾讯 API 数据源
- 洗盘概率模型

## V1.0 (2026 上半年)
- 单维度产业评分
- 人工选股，无系统性框架
