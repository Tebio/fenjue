# 焚诀 V2.3

> 焚诀的目标不是预测下一只牛股，而是建立一套可持续迭代的决策系统，在认知差出现时识别机会，在逻辑失效时及时退出，并通过持续回测和交易日志不断修正自身。
>
> 先研究产业，再寻找认知差；先判断逻辑，再设计交易；先定义失效条件，再决定是否持有。

**状态：封版冻结。此后只修 Bug 和微调参数，不增删框架。**

---

## 四层架构

```
Level 0: 市场状态 (Market Regime) → 决定仓位上限
   ↓
Level 1: 产业雷达 (Industry Radar) → 决定钱去哪
   ↓
Level 2: 公司映射 (Stock Mapping)  → 决定买谁（认知差×产业迁移）
   ↓
Level 3: 交易执行 (Trade Execution) → 决定怎么买、何时删
```

---

## Level 0: 市场状态

| 状态 | 仓位上限 | 触发条件 |
|------|:--:|------|
| Risk-On 激进 | 100% | 纳指新高 + A股放量 + 融资上行 |
| Risk-Neutral 中性 | 60% | 纳指震荡 + A股结构性 + 融资平稳 |
| Risk-Off 防御 | 30% | 纳指破位 + A股缩量 + 融资下降 |
| Crisis 避险 | 空仓 | 系统性风险，仅军工/红利避险 |

市场状态影响产业权重：Risk-Off 时 AI 材料降权、军工升权。

### 资金风格

| 风格 | 当前评级 | 含义 |
|------|:--:|------|
| 机构主导 | ★★★☆☆ | 适合中线持有，重基本面 |
| 游资主导 | ★★★☆☆ | 适合事件驱动，快进快出 |
| 量化主导 | ★★★★☆ | 适合高抛低吸，不追涨 |

## Level 2: 公司映射

### 六维评分公式

```
总分 = 产业趋势×0.35 + 资金流向×0.25 + 机构×0.10 + 融资×0.10 + 量化×0.05 + 预期兑现×0.15
```

最终总分 × 产业雷达权重。

### 产业映射树（不是行业分类，是资金逻辑）

```
AI材料
├── 磷化铟衬底 → 高纯磷(600141 兴发) / 锗(002428 云南锗业)
```

### 认知差指数

不是"公司好不好"，而是"市场有没有充分认识它"。兴发认知差 9（市场还当它是磷化工），长电认知差 2（全市场共识）。

## Level 3: 交易执行

每只标的附带 3 步建仓计划：初始建仓 → 确认加仓 → 完全建仓。每步含触发条件 + 仓位比例 + 价位区间。

### 失效条件

卖出不是因为跌了 15%，而是因为产业逻辑失效。每只标的定义专属失效条件。
