# 焚诀 — Investment Research Lab

> 不是炒股秘籍。是投资研究实验室。
>
> 目标不是证明自己永远正确，而是持续验证、修正和进化。

---

## 核心指标

不看单票，看体系。

| 指标 | 含义 |
|------|------|
| **Hit Rate** | S池选股跑赢沪深300的比例（目标 > 60%） |
| **Alpha** | 组合收益率 − 沪深300收益率 |
| **最大回撤** | 组合历史最大跌幅 |

## 四层架构

```
Level 0: 市场状态 → Risk-On / Neutral / Off / Crisis
Level 1: 产业雷达 → AI材料 ★★★★★ → 半导体 ★★★★ → ...
Level 2: 公司映射 → 认知差 × 产业映射树 → 六维评分 → S/A/B池
Level 3: 交易执行 → 建仓模型 + 失效条件
```

---

## 纪律

1. **每周日 Review** → Hermes 自动生成周报（各池收益、Hit Rate、Alpha）
2. **90% 时候不改公式** → 不因一次输赢调参数
3. **卖出是因为逻辑失效** → 不是跌了 15%
4. **Git 不撒谎** → 每份评分永久留痕，无法事后美化

---

## 原则

> 先研究产业，再寻找认知差；
> 先判断逻辑，再设计交易；
> 先定义失效条件，再决定是否持有。

> 一年后如果这个仓库里不是多了几十个新指标，而是多了 365 份评分、52 篇周报、12 次月度复盘，以及几次明确记录"我们判断错了，为什么错"——
> 那才是真的成功。
