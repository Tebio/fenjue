# 深V反转策略

## 条件
- 日内振幅 >7%
- 收盘位（日内位置）>70%（收盘在日内高位）
- 收阳（收盘 > 开盘）

## 昊华科技 600378 回测

6次信号，5红1绿 = **83%胜率**，均值 **+3.62%**。

| 日期 | 振幅 | 位 | 收 | 次日 |
|:--|:-:|:-:|:-:|:-:|
| 6/5 | 10.8% | 72% | +4.3% | **+7.8%** ✅ |
| 6/4 | 12.3% | 61% | +7.1% | **+7.0%** ✅ |
| 6/8 | 12.7% | 82% | +10.4% | **+5.1%** ✅ |
| 5/13 | 9.1% | 89% | +7.1% | **+4.0%** ✅ |
| 5/26 | 11.6% | 100% | +10.8% | **+1.9%** ✅ |
| 6/2 | 8.6% | 68% | +3.4% | -4.1% ❌ |

## 关键规则

- **收盘位>70%是关键分界线**：4/4全胜 vs 1/2（6/2位68%失败）
- **盘中位>70%不能作为入场信号**（6/2盘中可能到过70%，尾盘跌破）
- 必须等到14:50以后确认收盘能守住>70%

## 与策略B的区分

| | 深V | 策略B（快金叉+低开） |
|:--|:--|:--|
| 开盘 | 通常低开 | 低开 |
| 过程 | 砸→拉 | 低开后反弹 |
| 确认 | **收盘才能确认** | 9:25竞价即可判断 |
| 入场 | T+1 | **当日开盘** |
| 未来函数 | ⚠️ 收盘才知道位 | ✅ 无 |

深V策略比策略B晚了整整一天——策略B是当日开盘买，深V是收盘确认后次日买。

## 适用

个股级别策略，需单独回测。不同股票的深V胜率不同，不能跨票套用。
