# 数据源与容量

## 当前免费链路

- Sina 批量实时行情：主源。
- 腾讯批量实时行情：交叉校验和降级。
- 腾讯前复权日线：本地 SQLite 历史缓存。
- 日 K 每个代码每天最多增量刷新一次。
- 5 分钟线默认缓存 60 秒，同票并发自动合并。
- 实时、日 K、分钟线共享 6 路上游并发闸门。
- 上游失败时返回最近缓存并明确标记延迟，禁止伪装成实时数据。

20 并发共返回 400 条双源一致报价，无价格冲突。几十人群、实际十余人聊天、
每日十几到二十只股票的规模暂时不需要购买 488 元 API。

## 何时升级付费

满足任一条件再考虑 StockAPI 或券商行情：

- 连续 7 天可用率低于 99%。
- 20 并发 P95 长期超过 3 秒。
- 单源降级比例超过 5%。
- 需要分钟 K、逐笔、稳定复权或自动交易 SLA。

StockAPI 官网公开页没有列出 488 套餐的明确 QPS、日配额和历史深度。购买前应
先索取试用 token，并运行：

```bash
python -m fenjue benchmark --users 20 --symbols 20
```

不能只看宣传中的“毫秒响应”和“99.9%”。
